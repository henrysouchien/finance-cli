# Knowledge Base

This directory holds curated knowledge content the CashNerd agent retrieves from during reasoning. Each topic lives in a Markdown file with YAML frontmatter.

See [`docs/knowledge/CONTRACT.md`](../../../docs/knowledge/CONTRACT.md) for the file format and retrieval contract.
See [`docs/knowledge/TEMPLATE.md`](../../../docs/knowledge/TEMPLATE.md) when adding a new file.
