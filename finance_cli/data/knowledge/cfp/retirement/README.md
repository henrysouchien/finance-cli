# CFP Knowledge — Retirement Savings and Income Planning

Knowledge files for the CFP Principal Knowledge Domain "Retirement Savings and Income Planning" (`cfp_domains: [retirement]`).

- `retirement.retirement-accounts` - retirement account type and contribution vocabulary.
- `retirement.retirement-income-sources` - income-source inventory, data gaps, and handoff triggers.
- `retirement.social-security-claiming-readiness` - Social Security claiming vocabulary and professional-review boundaries.
- `retirement.medicare-enrollment-readiness` - Medicare timing inventory and SHIP/insurance handoff framing.
- `retirement.rmd-and-distribution-basics` - RMD and distribution vocabulary with tax-review boundaries.
- `retirement.pension-and-annuity-income-options` - pension and annuity option inventory without product/election advice.
- `retirement.withdrawal-order-education` - withdrawal sequencing education and CPA/fiduciary review framing.
- `retirement.retirement-income-readiness-handoff` - packet structure for retirement-income professional handoff.

See [`../../../../../docs/knowledge/CONTRACT.md`](../../../../../docs/knowledge/CONTRACT.md) for file format.
