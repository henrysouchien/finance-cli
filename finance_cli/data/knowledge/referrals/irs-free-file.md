# IRS Free File

IRS Free File is the official IRS access point for guided partner software offers. The coaching value is the routing discipline: start from IRS.gov, confirm the user's current-year eligibility and partner-specific state-return rules, and avoid commercial search-result paths that may not be the same Free File product.

Use this referral when the user has a relatively straightforward return and is comfortable self-filing with guided software.
